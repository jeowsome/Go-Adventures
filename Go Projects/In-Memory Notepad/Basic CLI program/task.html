Description

In this stage, you need to implement a basic CLI program that can read commands and exit when users provide a corresponding command.

Objectives

The program should run in an infinite loop. The cycle is the following:

  • The program asks users to input a command and data by printing Enter a command and data:

  • The program reads a line from the standard input stream and breaks it into two parts â€“ a command and data. Use a space (" ") as the separator between the command and the data parts;

  • If the command is exit, the program terminates and prints [Info] Bye!\n;

  • In all other cases, the program should print the given command, followed by the data, then print \n and go to the next iteration of the infinite loop.

Since the input line will be a whitespace-separated string like: "create This is my first record!" you'll need to use a Reader or a Scanner to properly read whitespace-separated strings from the standard input.

You can click the HINT button below to see a code example of using a Scanner along with functions of the strings package, and slice expressions to split the command and data.

Example

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Example 1: program execution

Enter a command and data: > create This is my first record!
create This is my first record!

Enter a command and data: > create This is my second record!
create This is my second record!

Enter a command and data: > list
list

Enter a command and data: > exit
[Info] Bye!
