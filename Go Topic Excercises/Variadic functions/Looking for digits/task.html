Looking for digits

In the first line of input, you will get a number. In the second line, you will get three digits. You need to write a variadic function that returns a slice with the digits from the second line of input that appeared in the number from the first line. You can have a look at the test cases for illustration.

Print the result of the function process in the output.


Sample Input:
1234
1 5 6

Sample Output:
[1]



Memory limit: 256 MB
Time limit: 8 seconds
