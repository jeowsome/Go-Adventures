Decomposing a function

\[|f| = \begin{cases} 2x \text{ if $x < 0$ } \\ 3x \text{ if $x > 0$} \end{cases}\]

Create two new functions f1, f2 for each case and complete the function f. The functions should each take one variable x as an argument and return a value.

Take notice that to solve this task, you must not delete or modify any code within the main function.

Sample Input:
-3

Sample Output:
-6



Memory limit: 256 MB
Time limit: 8 seconds
