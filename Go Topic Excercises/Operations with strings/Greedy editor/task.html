Greedy editor

Imagine that you are a text writer. Your new editor said he didn't want to pay for the void. Now he will pay 2 cents for each letter and 1 cent for the space symbol. You are lucky that you started learning Go. Now you can do it with literally 20 lines of code.

The easiest way to do this is to count all the letters in the text and then replace all the blank spaces " " with the empty character "", and then count them again.

Note that this method will only work with ASCII text. In it, all characters have a size of 1 byte. In other cases, you can use the utf8 package.

Finally, take notice that for this task, the input text only contains ASCII characters!


Sample Input:
One day, the noble Jacques, who lived in Vesteros, decided to move to another continent. But Vesteros didn't want to let him go.

Sample Output:
Symbols in text: 106
Spaces in text: 22



Memory limit: 256 MB
Time limit: 8 seconds
